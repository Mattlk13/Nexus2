## 🔐 Authentication

### Register
```http
POST /api/auth/register
Content-Type: application/json

{
  "email": "user@example.com",
  "password": "secure_password",
  "username": "creator_name"
}

Response:
{
  "token": "eyJhbGc...",
  "user": {
    "id": "user-123",
    "email": "user@example.com",
    "username": "creator_name",
    "role": "user"
  }
}
```

### Login
```http
POST /api/auth/login
Content-Type: application/json

{
  "email": "user@example.com",
  "password": "secure_password"
}

Response: (same as register)
```

### Get Current User
```http
GET /api/auth/me
Authorization: Bearer {token}

Response: { user object }
```

## 🔑 Authentication Notes

- Register/Login returns JWT token
- Include token in all protected routes: `Authorization: Bearer {token}`
- Token expires in 7 days
- Roles: `user`, `vendor`, `admin`

## 📦 Response Format

All endpoints return JSON with consistent structure:
- **Success**: `{ "success": true, "data": {...} }`
- **Error**: `{ "detail": "error message" }` (HTTP error codes)

## 🚦 Rate Limiting

- Standard: 100 requests/minute per IP
- Authenticated: 500 requests/minute per user
- AI Generation: 20 requests/minute per user

## 🌐 WebSocket Events (Socket.IO)

Connect: `wss://your-domain.com/api/socket.io`

**Events:**
- `connect` - Connection established
- `authenticate` - Send JWT for real-time notifications
- `notification` - Receive real-time notifications
- `new_follower` - Someone followed you
- `product_sold` - Your product was purchased

## 🎯 Quick Start Example

```javascript
// 1. Login
const authRes = await fetch('https://your-domain.com/api/auth/login', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    email: 'user@example.com',
    password: 'password'
  })
});
const { token } = await authRes.json();

// 2. Create AI Music
const musicRes = await fetch('https://your-domain.com/api/studio/music-generate', {
  method: 'POST',
  headers: {
    'Authorization': `Bearer ${token}`,
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    prompt: 'Upbeat electronic dance music',
    duration: 60
  })
});
const music = await musicRes.json();

// 3. Create Product Listing
const productRes = await fetch('https://your-domain.com/api/products', {
  method: 'POST',
  headers: {
    'Authorization': `Bearer ${token}`,
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    title: 'My AI Track',
    description: 'Electronic dance music',
    price: 4.99,
    category: 'music',
    file_url: music.audio_url
  })
});
```
